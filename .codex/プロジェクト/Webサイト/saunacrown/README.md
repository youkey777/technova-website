# Sauna Crown ブランドLP

## 概要
- サウナハットブランド「Sauna Crown」向けのビジュアル先行LP。
- 参考サイト(sauvenir.jp)の構成・動きを踏襲しつつ、オリジナルのHTML/CSS/JSで実装。

## ファイル
- `index.html`: プレーンHTML/CSS/JSで完結するレスポンシブLP。

## 表示方法
- 任意の静的ホスティング、またはローカルブラウザで `index.html` を開くだけで閲覧可能。
